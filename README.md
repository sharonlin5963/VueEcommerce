# 【Jailulu 宅家飾】電商網站
* [DEMO](https://sharonlin5963.github.io/VueEcommerce/#/)

## 簡介
### 
以 Vue.js練習開發SPA電商網頁。

## 網頁架構
### 
* 形象首頁
* 商品頁面
  * 總覽與分類
* 購物車系統
  * 新增、刪除、修改
* 結帳系統
* 會員登入
  * 帳號密碼驗證
  
## 使用技術
###
* Vue.js
* Vue CLI
* JavaScript ( ES6 )
* Scss
* Bootstrap 4 + RWD 製作響應式網頁
* Json-server + Heroku 開發 API

## 使用套件
###
* vue-axios 串接API
* Bootstrap 4
* Font Awesome
